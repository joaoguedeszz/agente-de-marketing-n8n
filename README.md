# outputs/

Os entregáveis do agente são salvos aqui, organizados por projeto:

```
outputs/
└── <nicho>-<data>/
    ├── 01-mineracao.md      ← ranking de ofertas escaladas (dados reais da Ad Library)
    ├── 02-validacao.md      ← teardown + viabilidade + GO/NO-GO
    ├── 03-criativos/        ← copy + prompts de imagem + imagens/vídeos gerados
    └── 04-funil.md          ← blueprint completo + matemática de validação
```

Exemplo: `outputs/emagrecimento-2026-06-07/`
